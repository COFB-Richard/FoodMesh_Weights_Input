Instructions

Number keys can be used to select food type.
Bin count can be incremented and decremented by the (q or -) and (w or +) keys respectively.
Press the "/" key to toggle compost.
Tab and (Shift + Tab) to move down an up the input boxes
Space Bar can be used to toggle the selected check box or press buttons
Press Enter to submit.

Removing a row will add it's values to the input fields.

Bin weight is not subtracted at any point yet 2023-12-06
